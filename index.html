suggested implementation
classifier
Universal InputClassifier
A single classify(text) called at the top of every phase handler. Returns one of: city | goal | type | persona | name | number | unknown. Prevents wrong data from silently polluting session fields.
CITY_SET = load_cities() # ~150 cities GOAL_KW = {"شراء","إيجار","بيع","rent","buy"} TYPE_KW = {"شقة","فيلا","أرض","مكتب","دور"} PERSONA_KW= {"مشتري","مستثمر","وكيل"} def classify(text: str) -> str: t = text.strip().lower() if t in GOAL_KW: return 'goal' if t in TYPE_KW: return 'type' if t in PERSONA_KW: return 'persona' if fuzzy(t, CITY_SET, cutoff=80): return 'city' if re.search(r'\d', t): return 'number' return 'unknown'
cross-phase
Answer Banking
When a user answers ahead of the current phase, store the answer in session_data['banked'] and skip that question later. Recovers ~30% of sessions that answer out-of-order.
def bank_answer(session, field, value): sd = session.session_data or {} sd.setdefault('banked', {})[field] = value session.session_data = sd session.save(update_fields=['session_data']) # In each phase handler: if field in session.session_data.get('banked', {}): session.field = banked[field] skip_question(); advance_phase()

Retry Counter per Phase
Track retries per phase in session. After max retries, use a sensible default and advance — never loop forever. Show buttons on the second retry instead of open text.
def retry_or_default(session, phase_key, default): n = session.session_data.get(f'retry_{phase_key}', 0) if n >= 2: session.session_data[phase_key] = default advance_phase(session) return True # used default session.session_data[f'retry_{phase_key}'] = n + 1 session.save(update_fields=['session_data']) return False
